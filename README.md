# CarPrice_Prediction
This dataset contains information about used cars.
This data can be used for a lot of purposes such as price prediction to exemplify the use of Random forest regressor in Machine Learning. The dataset is taken from kaggle.
The columns in the given dataset is as follows:

Car_Name,
Year,
Selling_Price,
Present_Price,
Kms_Driven,
Fuel_Type,
Seller_Type,
Transmission,
Owner.
